✈️ **Make Bee Trip**                                                                                                                 
A full-stack travel booking web application where users can explore and book Flights, Buses, Trains, and Hotels easily
**_______________________________________________________________________________________________________________________________**

.🚀 **Live Demo**

.🔗 https://make-bee-trip-wcjt.onrender.com

**_________________________________________________________________________________________________________________________________**

📌 **Features**
✈️ Flight booking UI
🚌 Bus booking UI
🚆 Train booking UI
🏨 Hotel listing system
🌐 REST API integration (Node + Express)
⚡ Fast and responsive frontend (React)
🎨 Modern UI design

**________________________________________________________________________________________________________________________________**


🛠️ **Tech Stack**
**Frontend**:React.js CSS Axios
**Backend:** Node.js,Express.js
**Database:** MongoDB

⚙️ **Installation & Setup**
1. Clone repo
 git clone https://github.com/your-username/make-bee-trip.git

2. **Install frontend**
    cd frontend
    npm install
    npm start
3. **Install backend**
cd backend
npm install
node server.js
🔗 **API Endpoint**
GET /api/hotels
